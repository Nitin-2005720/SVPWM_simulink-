# Space Vector Pulse Width Modulation (SVPWM) for Three-Phase VSI

## Overview
This project implements Space Vector Pulse Width Modulation (SVPWM) for a three-phase Voltage Source Inverter (VSI) using MATLAB/Simulink.

SVPWM is an advanced PWM technique used in motor drives and power electronic converters to achieve better DC bus utilization and lower harmonic distortion compared to conventional SPWM.

## Features
- Three-phase VSI model
- SVPWM switching signal generation
- Sector identification logic
- Dwell time calculation
- Gate pulse generation for inverter switches
- Output voltage waveform analysis

## Software Used
- MATLAB/Simulink

## File
- `SVPWM21 (1).slx` : Complete Simulink implementation of SVPWM for a three-phase inverter.

## Applications
- Motor Drives
- Renewable Energy Systems
- Grid-Connected Inverters
- Electric Vehicle Power Converters
